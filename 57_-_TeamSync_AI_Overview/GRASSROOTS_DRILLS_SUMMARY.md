## VLM Data Structure

Each drill includes comprehensive Visual Layout Metadata:

### Zones
- **Count**: 1-5 zones per drill (based on player count)
- **Properties**: ID, label, X/Y coordinates, width, height, color, fill opacity
- **Purpose**: Define training areas where players perform activities

### Player Positions
- **Active Players**: Distributed across zones
- **Bench Players**: Additional 3 players positioned off-field
- **Properties**: ID, label, X/Y coordinates, zone assignment, role

### Movement Patterns
- **Sequenced**: Timed at intervals (timestamps 0, 2, 4, 6...)
- **Mirrored**: Same pattern across different zones
- **Actions**: Dribbling, passing, running
- **Properties**: Path coordinates, duration, action type

### Equipment
- **Goals**: Placed for shooting/finishing drills
- **Cones**: Mark zone boundaries (4 corners per zone)
- **Properties**: ID, X/Y coordinates, type, label

### Metadata
- **Field Dimensions**: Age-appropriate field sizes
- **Generated**: True (indicates automated generation)
- **Source**: "grassroots-curriculum"
